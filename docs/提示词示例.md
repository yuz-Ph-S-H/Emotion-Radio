# 共感电台提示词示例（课堂展示版）

> 本文所有"实测效果"均由本地分析器（`LocalAnalyzer`）真实运行得到，
> 标注格式为 `效价/能量 → 调式 · 背景床 · 主要粒子`。
> 效价越低越偏小调暗色、旋律下行；越高越偏大调亮色、旋律上行。

## 使用说明

1. 双击 `run.bat` 启动，浏览器打开 `http://127.0.0.1:5000`。
2. **进入页面后先点一下页面**（浏览器要求交互后才能出声），再输入文字。
3. 在底部输入框粘贴下面任意一句，按回车即可。

## ⭐ 黄金演示路线（推荐课堂逐句输入）

一条从低落到明亮再到梦幻的情绪叙事，**音乐与画面随情绪连续变化**，最能体现"文字→情感→视听"的映射。

每句会被判定为一种**情绪性格**，从而采用不同的旋律句式、配器与节奏（共 6 种性格，刚好一句一种）。

| 顺序 | 输入提示词 | 实测效果（性格 → 调式 · 句式/配器 · 粒子） |
| --- | --- | --- |
| 1 | 窗外是冰冷的小雨，房间里只剩回忆和叹息。 | 0.38/0.32 → **忧伤** · 小调(2716Hz暗) · 缓慢下行·钢琴/钟琴(无鼓) · 微光/星钻 |
| 2 | 深夜的雨一直落下，我孤独地走在回家的长路上。 | 0.49/0.37 → **平静** · 大调 · 连绵句式 · 夜色 · 细雨/漂浮 |
| 3 | 清晨的薄雾散开，小溪在山谷里安静流动。 | 0.64/0.34 → **平静** · 大调 · 长笛/钟琴 · 阳光/流水 |
| 4 | 阳光照进花园，蝴蝶在花间快乐飞舞。 | 1.00/0.43 → **温柔** · 五声大调(6300Hz亮) · 歌唱句式·暖吉他+点缀 · 阳光/花瓣 |
| 5 | 城市上空升起绚烂的烟花，照亮所有笑容。 | 1.00/0.58 → **欢快** · 五声大调 · 跑动琶音+鼓点 · 烟花/星钻 |
| 6 | 梦境里出现极光，光芒在天空缓慢流动。 | 0.99/0.45 → **梦幻** · 五声大调 · 宽广琶音·钟琴/铺底 · 极光 |

**讲解提示**：
- 第 1 句（忧伤·小调·下行·无鼓）与第 5 句（欢快·大调·琶音·有鼓）反差最强，最适合开场对比。
- 提醒观众：旋律是**乐句**生成（有起伏、变形与留白），不是同一段循环；不同性格连鼓点、乐器数量都不同。

## 🎯 对比演示：同样的"雨"，不同的情绪

用来直观说明系统不是按关键词查表，而是综合**情感效价**生成音乐。

| 输入提示词 | 实测效果 | 说明 |
| --- | --- | --- |
| 窗外是冰冷的小雨，房间里只剩回忆和叹息。 | 0.38 → **小调 · 暗** | 同样有"雨"，情绪低落 → 忧伤 |
| 雨停以后出现彩虹，阳光照进花园，远处还有雷声。 | 0.86 → **大调 · 亮** | 同样有"雨"，但整体明亮 → 温暖 |

> 两句都含意象"雨"，音乐走向却截然相反，体现意象效价先验 + 情绪融合（见 `数学推导说明.md` §3）。

## 情绪分组示例（均附实测效价）

### 忧伤 / 孤独（小调、暗色、旋律下行）

- 窗外是冰冷的小雨，房间里只剩回忆和叹息。 — `0.38 小调`
- 秋天的落叶铺满街道，离别让人感伤。 — `0.41 小调`
- 黑夜很安静，月光落在窗边，眼泪也变得很轻。 — *偏消极*

### 紧张 / 震撼（小调、高能量、风暴粒子）

- 暴雨和狂风席卷城市，雷声让夜晚震动。 — `0.38/0.66 小调 · 风暴/闪电`
- 烈火在旷野里燃烧，火花不断飞向天空。 — *高能量 · 火焰/星钻*
- 海浪撞向悬崖，暴风把天空压得很低。 — *高能量 · 风暴*

### 平静 / 治愈（大调、柔和、低能量）

- 清晨的薄雾散开，小溪在山谷里安静流动。 — `0.64 大调 · 流水`
- 月光洒在窗边，心里一片安宁。 — `0.67 大调 · 月光`
- 湖面泛起涟漪，微风温柔地吹过竹林。 — `0.98 五声大调 · 微风/涟漪`

### 欢快 / 明亮（五声大调、暖色、旋律上行）

- 阳光照进花园，蝴蝶在花间快乐飞舞。 — `1.00 · 阳光/花瓣`
- 春风吹过桃花，笑容像露珠一样发亮。 — `0.97 · 微风/花瓣`
- 城市上空升起绚烂的烟花，照亮所有笑容。 — `1.00/0.58 · 烟花`

### 梦幻 / 辽阔（五声大调、空旷混响、夜色铺底）

- 星空铺满夜晚，银河像一条安静的河流。 — `0.74 · 星光/流动`
- 梦境里出现极光，光芒在天空缓慢流动。 — `0.99 · 极光`
- 萤火虫在森林里亮起，像漂浮的星星。 — *萤火微闪*

## 🎼 混合场景：多意象 → 多声部 + 多粒子

一句话里放多个意象，系统会按权重分配主旋律、和声、点缀音色，并叠加多种粒子。

> **雨停以后出现彩虹，阳光照进花园，远处还有雷声。**

实测：`0.86/0.56` → 五声大调，识别意象 **雨 / 彩虹 / 阳光 / 花园 / 雷声**，
音色 `鼓点 + 钟琴 + 吉他 + 钢琴` 四声部，粒子 `闪电 / 彩虹 / 阳光 / 雨 / 花瓣` 同时出现。

适合用来展示"复合情绪 → 多层次视听"的能力。

## 👥 观众互动备用句

现场可邀请观众说一句话即时生成，下面几类几乎都能触发明显效果：

- 一种天气：今天下着雨 / 外面是大晴天 / 风很大。
- 一个地方 + 心情：回到家，心里很安宁 / 一个人在海边，有点孤独。
- 一句当下心情：今天很累，但我还想看见一点光。
- 一句诗意短句：月光落在窗台，像一层薄薄的雪。

> 小技巧：句子里带上**情绪词**（孤独、温暖、喜悦、绝望）能让效价更鲜明，反差更大。

## 讲解要点（答辩可直接照念）

1. **不是关键词查表**：相同意象"雨"在不同情绪下生成截然不同的音乐（见对比演示）。
2. **情绪是连续量**：效价/能量连续地驱动调式、音色亮度（低通截止）、旋律走向、和弦色彩，
   不是几个固定档位（推导见 `数学推导说明.md` §16~§22）。
3. **三级情感融合**：SnowNLP 整体语气 + 意象效价先验 + 情绪关键词修正（§3），
   解决了 SnowNLP 对中文整体偏积极、导致"什么都是大调"的问题。
4. **听感优化**：背景音乐为本地原创合成（无版权问题），经升余弦音头消爆音、低通柔化、
   混响与无缝循环处理，并通过母带限幅杜绝破音。

## API 增强方案说明

切换到 API 方案（`config.py` 中 `NLP_MODE="api"`）后，效价/能量/意象由大语言模型直接给出，
不再经过本地三级融合（§3 末注）。系统提示词见 `nlp/api_analyzer.py` 的 `SYSTEM_PROMPT`，
要求模型仅输出包含 `valence`、`arousal`、`imageries`、`imagery_weights`、
`suggested_instrument`、`suggested_effects` 等字段的 JSON。本地方案无需任何密钥即可演示。
